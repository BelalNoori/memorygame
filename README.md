Memory Game

Ein Memory-Kartenspiel gebaut mit TypeScript und Vite.

Features

- 2 Spieler – Blau vs. Orange, abwechselnd am Zug
- 2 Themes – Code Vibes (dunkel) & Food Theme (hell/orange)
- 3 Schwierigkeitsstufen – 16, 24 oder 36 Karten
- Flip-Animation – 3D Kartenumdreh-Effekt
- Match-Highlighting – gematchte Karten werden markiert
- Scoreboard – Live-Score während des Spiels
- Game Over & Winner Screen – mit Übergangsanimation

Tech Stack

TypeScript – Spiellogik & DOM-Manipulation
Vite – Build Tool & Dev Server
SCSS – Styling mit Variablen, Mixins & Theme-Klassen
localStorage – Einstellungen & Scores zwischen Seiten

Du hast lust auf Memory, ich zeig dir, wie du es installieren und starten kannst.

npm install

npm run dev

npm run build
